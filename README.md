# 🏠 Smart Home Monitoring System (نظام مراقبة المنزل الذكي)

## 🛠️ بنيان النظام والأجهزة (System Architecture)
المشروع يعتمد بشكل أساسي على متحكم **Arduino** وبنية الحساسات الموزعة كالتالي:
* **Arduino IDE:** بيئة التطوير الأساسية المستخدمة لبرمجة وكتابة الأكواد بـ .
* **DHT & Smoke Sensors (`DHT_SMOKE`):** لنظام الأمان ومراقبة درجات الحرارة، الرطوبة، وكشف الدخان والغاز(Sensor Suite).
* **RFID System (`RFID` / `RFID_Reader`):** نظام الدخول الذكي والتحكم في الأبواب عبر الكروت الرقمية المشفرة.

## 💻 تعليمات التثبيت والتشغيل (Instructions for installation)
لمراجعة وفحص الأكواد البرمجية بشكل صحيح على جهازك دون الحاجة لتوصيل أي قطع إلكترونية (Hardware):
1. قم بتحميل المجلد بالكامل من رابط **Google Drive** المرفق أعلاه.
2. تأكد من تحميل وتثبيت بيئة التطوير **Arduino IDE** على جهازك لاستعراض الأكواد.
3. افتح ملفات الكود التي تنتهي بامتداد `.ino` المتواجدة داخل مجلدات الحساسات مباشرة باستخدام برنامج Arduino.
4. لتفادي أخطاء التجميع (Compilation Errors)، يرجى التأكد من تنصيب المكتبات الخاصة بالحساسات المستخدمة من خلال الـ Library Manager داخل البرنامج.
